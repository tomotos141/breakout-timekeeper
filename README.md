# ブレイクアウトルーム タイムキーパー

Zoomブレイクアウトルームでのグループディスカッション用タイムキーパーアプリ

## デモ

https://tomotos141.github.io/breakout-timekeeper/

## 機能

- **自動進行タイマー**: 設定時間が終わると自動で次のセクションへ
- **ピクチャーインピクチャー**: 小さなフローティングウィンドウで他のタブを見ながら確認可能
- **進捗表示**: パンくずリストで全体の進行状況を可視化
- **音声アラート**: 30秒前・10秒前・終了時にビープ音
- **準備フェーズ ON/OFF**: 2回目以降のMTGでは①をスキップ可能

## セッション構成

| セクション | 時間 |
|-----------|------|
| ① タイムキーパー・発表順の決定 | 2分 |
| ② 会社での役割 | 3分 |
| ③ 相談事項の共有 | 5分 |
| ④ チームメイトからのコメント | 10分 |
| ⑤ 学び・気づき/babystepの共有 | 3分 |

※ ②〜⑤を3人分繰り返し（計65分）

## 使い方

1. [デモページ](https://tomotos141.github.io/breakout-timekeeper/)を開く
2. Zoomで画面共有、またはミニ表示ボタンでフローティングウィンドウを開く
3. スタートボタンを押して開始

## ローカルで実行

```bash
# クローン
git clone https://github.com/tomotos141/breakout-timekeeper.git

# ブラウザで開く
open index.html  # macOS
start index.html # Windows
```

ビルド不要。ブラウザで直接開くだけで動作します。

## 動作環境

- Chrome 116以降（ピクチャーインピクチャー機能使用時）
- その他のブラウザでも基本機能は動作
